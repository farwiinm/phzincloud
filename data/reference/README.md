# data/reference/

This folder contains pKa reference databases used by the lookup module.

## Files

### PKAD-R-v1.0\_...csv

- **Source:** https://pkad.lab.clemson.edu/
- **Maintainer:** Clemson University
- **Contents:** Experimentally measured pKa values for titratable residues
  across known protein structures. Used as Tier 1 (experimental) source
  in pka_lookup.py.
